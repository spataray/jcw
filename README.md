# Johnny Castaway - Web Homage (JCW)

A modern, web-based homage to the classic 1992 Sierra screensaver, "Johnny Castaway".

![Johnny Castaway Web](https://via.placeholder.com/800x400?text=Johnny+Castaway+Web+Homage)

## 🏝️ Project Overview

Johnny Castaway Web is a nostalgic, ambient screensaver experience designed for modern web browsers, smart TVs, and car infotainment systems. It features a dynamically animated island where the character "Johnny" lives out his days (and nights) based on a randomized state machine and the local system time.

## ✨ Key Features

*   **Dynamic Island Scene:** CSS/SVG-based visuals for resolution independence and sharp display on high-DPI screens.
*   **Johnny's State Machine:** Randomized behaviors including:
    *   `IDLE`: Johnny gazes at the horizon.
    *   `SLEEPING`: Johnny takes a nap (more frequent at night).
    *   `EXERCISING`: Johnny does jumping jacks to stay fit.
    *   `DREAMING`: Johnny's thought bubbles reveal his inner desires (food, beer, escape).
*   **Day/Night Cycle:** The sky and ocean colors automatically transition based on the user's local system time (Day, Sunset, Night).
*   **Ambient Audio:** Gentle ocean waves and seagull cries (user-activated via start screen).
*   **CRT Aesthetics:** Subtle scanline and flicker effects for an authentic retro feel.

## 🚀 Getting Started

Since this is a static web application, it requires no build process.

1.  Clone the repository:
    ```bash
    git clone https://github.com/spataray/jcw.git
    ```
2.  Open `index.html` in any modern web browser.
3.  Click the overlay to start the animation and audio.

## 📂 Project Structure

*   `index.html`: The main entry point and UI structure.
*   `style.css`: All visual styling, animations, and CRT effects.
*   `script.js`: The "brain" of the project—state management and environmental control.
